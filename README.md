A one-paragraph domain description (see below)

Our project simulates a community emergency resource coordination system that connects residents, emergency responders, shelters, food banks, healthcare providers, and volunteers during natural disasters and other large-scale emergencies. The platform enables users to report urgent needs, locate available shelters and essential resources, request assistance, and coordinate volunteer efforts in real time. During events such as hurricanes, floods, or wildfires, a single server would become insufficient because thousands of users may simultaneously access the system to submit requests, update resource availability, and retrieve critical information, requiring a scalable distributed system to remain reliable under heavy demand. From a Computing for the Common Good perspective, this system directly benefits disaster-affected communities—particularly low-income families, older adults, people with disabilities, and emergency response organizations—by improving access to timely and accurate information when every minute matters. If the system performs well, emergency resources can be distributed more efficiently and reach those in need more quickly; if it fails or becomes unavailable, delays and misinformation could prevent vulnerable individuals from receiving life-saving assistance.
